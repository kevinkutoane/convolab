# ConvoLab Platform and Studio v1.0.0-alpha.11

This release stabilizes the complete functional v1 baseline before Workspace, Identity and Access Control begins.

The release adds route and CSS code splitting, hard bundle budgets, PostgreSQL migration and restart verification, cross-capability deterministic acceptance, Playwright smoke coverage, production placeholder removal, consistent lifecycle routing, reusable UI states, and version/encoding/ZAR release checks.

Compatibility remains intact for `/api/evaluation/*`, `/evaluation`, and all previously published capability contracts. The canonical expanded Evaluation contracts remain under `/api/evaluations/*` and `/evaluations`.

Beta is intentionally deferred until authentication, tenant isolation, role enforcement, managed environments, secret governance, audit, monitoring, and backup/restore controls are complete.
